Rules

Mastermind is a game where you have to guess the colour combination of the computer. You have 5 chances to guess the correct combination. After each guess, the computer will give you a hint. A 'bull' means that you have guessed the correct colour in the correct position. A 'cow' means that you have guessed the correct colour but in the wrong position. The possible colours in the code are as follows: red, green, blue, white, purple, and each should be entered as the first letter of the colour e.g. 'r' would be 'red'. Good luck!
